# Project Planning Phase

## Project Title
Human Development Index (HDI) Predictor

## Objective
To develop a Machine Learning web application that predicts the Human Development Index (HDI) of a country based on development indicators such as Life Expectancy, Mean Years of Schooling, Expected Years of Schooling, and Gross National Income (GNI) per capita.
